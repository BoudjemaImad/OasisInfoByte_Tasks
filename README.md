## Projects Included

### 1️⃣ Voice Assistant (Beginner Level)
**Description:**  
A basic command-line voice assistant that responds to simple voice commands.

**Features:**
- Responds to greetings (e.g., "Hello")
- Tells the current time and date
- Performs basic web searches based on user queries

**Key Concepts Used:**
- Speech recognition
- Python libraries for voice input/output
- Conditional logic
- Error handling

---

### 2️⃣ BMI Calculator (Beginner Level)
**Description:**  
A command-line BMI calculator that computes Body Mass Index based on user input.

**Features:**
- Accepts weight (kg) and height (m)
- Calculates BMI using the standard formula
- Classifies BMI into health categories:
  - Underweight
  - Normal
  - Overweight

**Key Concepts Used:**
- User input handling
- Mathematical calculations
- Conditional statements
- Input validation

---

### 3️⃣ Random Password Generator (Beginner Level)
**Description:**  
A command-line tool that generates random passwords based on user-defined criteria.

**Features:**
- User-defined password length
- Option to include:
  - Letters
  - Numbers
  - Symbols
- Generates secure random passwords

**Key Concepts Used:**
- Randomization
- String manipulation
- User input validation
- Python standard libraries

---

### 4️⃣ Basic Weather App (Beginner Level)
**Description:**  
A command-line weather application that fetches real-time weather data using an API.

**Features:**
- User-input location (city)
- Displays:
  - Temperature
  - Humidity
  - Weather conditions

**Key Concepts Used:**
- API integration
- JSON data parsing
- HTTP requests
- Error handling
